RFQ Submission System – HTML Documentation

System Type: QR-enabled, Email-based RFQ Submission (No Database)

---

1. System Overview

This RFQ system allows users to submit Request for Quotation details via a public HTML web form (accessed through a QR code). Once submitted:

  • All RFQ details are sent directly to the procurement email
  • Artwork and documents are attached to the email
  • User can review the submitted RFQ
  • User can print or save the RFQ as PDF
No database is used. All data is transient and delivered via email.
---

        ↓

        ↓
